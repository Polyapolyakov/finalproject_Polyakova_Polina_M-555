# finalproject_Polyakova_Polina_M-555
Консольное приложение, имитирующее работу валютного кошелька

# описание:
Пользователю предоставлено консольное приложение, имитирующее работу валютного кошелька

# установка зависимостей:
make install

# запуск приложения:
make valutatrade

# последовательность операций после запуска приложения:
После запуска приложения пользователю необходимо:
1) зарегистрироваться _(команда register)_.
2) осуществить вход _(команда login)_.

После регистрации и входа пользователю доступны команды: операции с кошельком: deposit, portfolio, buy, sell.
Описание всех реализуемых команд с примерами осуществляется после запуска приложения.

# валюты:
Пользователю доступны операции в следующих валютах: USD, RUB, EUR и криптовалютах: BTC, ETH.
Базовая валюта приложения - USD.
